# Analiza danych uczelni z użyciem LINQ

## Opis
Projekt konsolowy w C#, którego celem jest analiza danych uczelni z użyciem LINQ na obiektach w pamięci.  
Aplikacja wspiera pracę koordynatora programu, umożliwiając szybkie filtrowanie, wyszukiwanie i prezentowanie informacji o studentach, przedmiotach, prowadzących oraz zapisach.

## Zakres projektu
W projekcie zostały zaimplementowane metody wykonujące zapytania LINQ dotyczące:
- studentów,
- przedmiotów,
- prowadzących,
- zapisów.

## Wykorzystane elementy LINQ
W projekcie zostały użyte m.in.:
- `Where`
- `Select`
- `OrderBy`
- `ThenBy`
- `GroupBy`
- `Count`
- `FirstOrDefault`
- `Any`

## Cel ćwiczenia
Celem ćwiczenia było nauczenie się:
- pracy na kolekcjach obiektów,
- tworzenia zapytań LINQ,
- filtrowania i sortowania danych,
- prezentowania wyników w aplikacji konsolowej.
